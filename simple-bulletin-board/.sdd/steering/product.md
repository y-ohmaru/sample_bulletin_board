# Product Overview

## プロダクト概要
Spring Boot、H2 Database、Thymeleaf を用いた簡易掲示板アプリです。.sdd/description.md では、投稿の一覧表示、登録、削除を行う Web アプリケーションとして定義されています。

## 主要機能
- 投稿一覧表示: `GET /posts` で投稿一覧を表示する
- 投稿登録: 名前と内容を入力して投稿を登録する
- 投稿削除: 一覧画面から投稿を削除する
- バリデーション: 名前必須、内容必須、内容100文字以内、エラー時のメッセージ表示

## 対象ユースケース
投稿者がブラウザから掲示板にアクセスし、既存の投稿を確認しながら、新しい投稿の追加や不要な投稿の削除を行うシナリオを対象としています。

## 価値提案
フォーム入力、一覧表示、削除、バリデーション、永続化を含む基本的な CRUD の流れを、Spring Boot の標準的な構成で学習・実装できる点が価値です。アプリ起動時にテーブルを自動作成する要件があり、ローカルで扱いやすい構成になっています。
